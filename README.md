# Browser Automation Bot

This project is a simple Python browser automation bot built with Selenium.

It opens a webpage, reads the page title, extracts quotes from the page, and saves the results into a text file.

## What the program does

- opens a browser window
- goes to a webpage
- reads the page title
- finds elements on the page
- saves the extracted data into `browser_bot_output.txt`

## Technologies used

- `selenium`
- `webdriver-manager`

## Source website

The bot uses:

https://quotes.toscrape.com

This website is designed for practicing browser automation and scraping.
